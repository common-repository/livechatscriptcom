== Description ==

This plugin/widget  allows you to insert livechat script easily to your WorPress web pages.
Having the script enables you to display a button or image that your web page visitors can click so they can chat with you.
Livechat script connects to the free livechat service and is offered free for life (no catch) by www.livechatscript.com.

It has the following features:

*  Multiple chats in tabs
*  Unlimited simultaneous chats
*  Customizable images and text messages 
*  Customer IP/Country/City Location
*  Canned responses
*  Chat Queue display
*  Support international characters 
*  Spelling check
*  Typing Notification
*  Offline Message Email/ Notification
*  Works with most popular browsers 
*  Works with smart mobile phones, tablets
*  Works with PC, Linux , MAC
*  SSL secured message

and many more.

[Visit the author's website to find out more.](http://www.livechatscript.com)

== Installation ==

1. Download and extract the zip archive
2. Upload `enhanced-text-widget` folder to `/wp-content/plugins/`
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Add the widget to a sidebar . Configure the widget by pasting the livechat script.
